# Privacy Policy (AMO)

IZW Prayer Times (Vienna) is designed to work offline and privately.

## What data is stored
The extension stores only user settings, such as:
- notification enabled/disabled
- notification lead time (minutes)
- badge enabled/disabled
- Hijri date correction value
- language preference (auto/English/Arabic)
- Ramadan theme debug override

## Where data is stored
All settings are stored locally in your browser's extension storage (`storage.local`).

## Data collection and sharing
- We do **not** collect personal data.
- We do **not** transmit your data to any server.
- We do **not** use analytics, advertising SDKs, or trackers.
- We do **not** sell, rent, or transfer user data.

## Network access
The extension does not require external network access for its core functionality. Prayer times are bundled in the extension package.
