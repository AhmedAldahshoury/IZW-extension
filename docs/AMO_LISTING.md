## Add-on name
IZW Prayer Times (Vienna)

## Short summary
Offline Vienna prayer times from IZW with Hijri date, Ramadan theme, Jumu'ah highlight, reminders, and Arabic/English support.

## Full description
IZW Prayer Times (Vienna) is a lightweight, privacy-focused Firefox extension that shows daily prayer times and the next upcoming prayer.

Features:
- Offline IZW Vienna prayer timetable (bundled data)
- Next prayer with live countdown
- Hijri date with correction setting
- Automatic Ramadan visual theme (plus optional debug override)
- Friday (Jumu'ah) highlight for Dhuhr row
- Prayer reminder notifications (optional)
- Toolbar badge countdown (optional)
- Arabic and English interface

Privacy-first design:
- No analytics
- No trackers
- No remote scripts
- No data collection
